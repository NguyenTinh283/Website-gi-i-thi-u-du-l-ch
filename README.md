# Website Giới Thiệu Du Lịch

## 📌 Giới thiệu
Website giới thiệu các địa điểm du lịch và các món ăn nổi tiếng. 
Dự án được xây dựng nhằm thực hành thiết kế giao diện và xử lý dữ liệu bằng PHP.

## 🛠 Công nghệ sử dụng
- PHP
- HTML
- CSS
- MySQL

## 🎯 Chức năng chính
- Hiển thị danh sách địa điểm du lịch và món ăn
- Xem chi tiết địa điểm và món ăn
- Xem bản đồ
- Quản lý dữ liệu bằng cơ sở dữ liệu MySQL
- Tích hợp giao diện thân thiện với người dùng
